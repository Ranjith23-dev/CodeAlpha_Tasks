# RekShoppe Store

A full-stack e-commerce web application built with Django, designed to provide a seamless online shopping experience. The platform features product browsing, detailed product pages, shopping cart functionality, and an intuitive user interface for efficient product management and purchasing workflows.

### Key Features

* Product catalog with detailed product information
* Shopping cart management
* Responsive and user-friendly interface
* Django-powered backend architecture
* Dynamic product rendering using Django templates
* Static file management for styling and assets
* Database integration for product storage and retrieval
* Scalable project structure following Django best practices

### Technology Stack

* Backend: Django (Python)
* Frontend: HTML, CSS, JavaScript
* Database: SQLite
* Version Control: Git & GitHub

### Project Objective

This project demonstrates the implementation of a modern e-commerce platform while applying full-stack web development principles, database management, and backend integration using Django. It showcases practical skills in web application development, software architecture, and user-centric design.
